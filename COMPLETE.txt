FIREBASE DATA STRUCTURE
================================================================================

Stored in Firestore under:
  users/{uid}/
    collection/{paintId}       → { owned: true, notes: "..." }
    palettes/{paletteId}       → { name: "...", paintIds: [...], createdAt: "..." }
    recipes/{recipeId}         → { name: "...", miniature: "...", steps: [...], createdAt: "..." }

All data syncs automatically and is encrypted in transit.

Firebase Project: palette-67fd6
Auth Method: Google Sign-In
Database: Firestore (Real-time)

================================================================================
COLOUR-BLIND ACCESSIBILITY
================================================================================

This app is specifically designed for colour-blind users:

  ✓ Paint descriptions in plain English (not hex codes)
    Example: "warm mid-tone red", "cold dark blue-grey"
  
  ✓ Colour family categories (Reds, Greens, Cool Greys, Metallics, etc.)
  
  ✓ High contrast UI (dark text on light backgrounds)
  
  ✓ No colour-only information (all info conveyed by text + colour)
  
  ✓ Large, touch-friendly buttons (48x48px minimum)
  
  ✓ Clear labels on all form inputs

Colour swatches are shown for sighted reference but are NOT the primary
information. The description and name are always prominent.
