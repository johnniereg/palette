================================================================================
                        KEY FEATURES
================================================================================

PAINT COLLECTION BROWSER
  Browse all 85 paints with detailed information
  Filter by: brand, type, colour family, ownership
  Search by paint name
  Mark paints as owned/unowned
  Add personal notes (e.g., "thin 2:1", "use over grey primer")
  All data syncs to Firebase automatically

COLOUR PALETTES
  Create named palettes ("Ork Skin", "Rusted Metal", etc.)
  Add paints to palettes
  View shopping list of paints you don't own
  Mark items as purchased directly from palette
  Organize painting projects by colour scheme

PAINTING RECIPES
  Create step-by-step painting guides
  Each step includes:
    - Paint selection from database
    - Technique notes (mixing ratios, application method)
    - Visual colour swatch
  Full editing and deletion support

COLOUR-BLIND ACCESSIBILITY
  Plain English descriptions (primary information)
    Example: "warm mid-tone red" instead of "#A41E27"
  Colour family tags (Reds, Greens, Cool Greys, Metallics, etc.)
  High contrast UI (dark text on light backgrounds)
  Large, easy-to-tap buttons
  No colour-only information

================================================================================
                        DEPLOYMENT (5 MINUTES)
================================================================================

To deploy to GitHub Pages:

1. Create a GitHub repository:
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/palette.git
   git push -u origin main

2. Enable GitHub Pages:
   - Go to repository Settings → Pages
   - Under "Build and deployment"
   - Select "Deploy from a branch"
   - Choose "gh-pages" branch
   - Click Save

3. GitHub Actions deploys automatically:
   - Triggered on push to main branch
   - Builds the project (npm run build)
   - Deploys to gh-pages branch
   - Takes 2-3 minutes
   - Your app is live at: https://YOUR_USERNAME.github.io/palette/

No additional configuration needed. Everything is already set up!
